# Simple tic-tac-toe on Haskell language

> [!WARNING]  
> The algorithm of computer player is very ingenious

The game has two modes:
* Player vs Player
* Player vs Computer

The game process looks like:
```shell
Enter game mode (1 - Player vs Player. 2 - Player vs Computer): 2
Your(X) move: 
 | | 
-----
 | | 
-----
 | | 
Enter row number (1-3): 2
Enter column number (1-3): 2
Computer make move!
Your(X) move: 
O| | 
-----
 |X| 
-----
 | | 
Enter row number (1-3): |
```
